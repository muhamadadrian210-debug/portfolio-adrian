🔒 Keamanan Web

Standar Keamanan Website Ini

Website ini dibangun dengan menerapkan standar keamanan web modern yang direkomendasikan oleh OWASP dan W3C, memastikan setiap interaksi pengguna terlindungi dengan baik.

✅ Website ini telah menerapkan standar keamanan web modern — aman diakses dari perangkat apapun. Proyek Sivilize Hub Pro dan SiKasir yang saya bangun juga dilengkapi firewall server-side nyata dengan DDoS & Brute Force Protection.
🛡️

XSS Protection

Semua output dinamis di-escape menggunakan fungsi escapeHtml() untuk mencegah serangan Cross-Site Scripting (XSS) yang dapat menyuntikkan kode berbahaya ke halaman.

🔗

Tab Hijacking Prevention

Semua tautan eksternal menggunakan atribut rel="noopener noreferrer" untuk mencegah halaman tujuan mengakses atau memanipulasi halaman asal (reverse tabnapping attack).

🔐

HTTPS Ready

Website dirancang untuk berjalan di atas protokol HTTPS dengan enkripsi TLS/SSL, memastikan semua data yang dikirim antara browser dan server terenkripsi dan tidak bisa disadap.

🚫

Zero Data Exposure

Tidak ada data sensitif, kredensial, API key, atau informasi backend yang terekspos di sisi klien. Semua konten bersifat publik dan aman untuk ditampilkan.

♿

WCAG 2.1 AA Compliance

Website memenuhi standar aksesibilitas WCAG 2.1 level AA, termasuk rasio kontras warna minimum 4.5:1, navigasi keyboard penuh, dan atribut ARIA yang lengkap.

🏗️

Semantic HTML Structure

Menggunakan elemen HTML semantik (<header>, <main>, <section>, <footer>) yang membantu browser dan screen reader memahami struktur halaman dengan benar.

🖼️

Safe Image Handling

Setiap elemen gambar memiliki atribut alt deskriptif dan fallback handler yang aman. Gambar yang gagal dimuat digantikan placeholder tanpa error yang bisa dieksploitasi.

⌨️

Keyboard Navigation Security

Implementasi skip link dan focus management yang aman mencegah focus trap attack. Semua elemen interaktif dapat diakses via keyboard dengan urutan tab yang logis dan aman.

📱

Responsive & Safe Rendering

Viewport meta tag dikonfigurasi dengan benar untuk mencegah zoom manipulation attack pada perangkat mobile. Layout responsif memastikan konten ditampilkan dengan aman di semua ukuran layar.

🎭

Reduced Motion Support

Menghormati preferensi sistem operasi pengguna dengan menonaktifkan animasi saat prefers-reduced-motion aktif, melindungi pengguna dengan kondisi vestibular disorder.

⚡

Performance Security

Menggunakan loading="lazy" untuk gambar non-kritis dan rel="preconnect" untuk resource eksternal, mencegah resource exhaustion dan menjaga performa tetap optimal.

🔥

Firewall Nyata di Proyek Sivilize Hub Pro

Sebagai bukti kemampuan keamanan saya, proyek Sivilize Hub Pro yang sudah live di sivilize-hub-pro.vercel.app dilengkapi sistem keamanan server-side yang sesungguhnya:

🛡️
DDoS Protection Membatasi max 200 request per menit per IP, memblokir otomatis selama 30 menit jika terdeteksi serangan
🔒
Brute Force Prevention Maksimal 5 percobaan login gagal dalam 15 menit, akun diblokir 1 jam jika melebihi batas
⚡
Aggressive DDoS Detection Mendeteksi pola serangan cepat (100+ request dalam 5 detik), blokir otomatis 2 jam
🌐
Concurrent Connection Limit Membatasi maksimal 20 koneksi bersamaan per IP untuk mencegah resource exhaustion
🚨
Suspicious Pattern Detection Memantau pola mencurigakan seperti 404 berulang dan error rate tinggi secara real-time
📋
IP Blacklist Permanen IP yang terbukti melakukan serangan diblokir secara permanen dari seluruh sistem
Arsitektur Keamanan

11 Layer Perlindungan

Sistem keamanan berlapis yang memastikan setiap ancaman ditangani sebelum mencapai data pengguna.

01
🔐

Security Response Headers

Inject header keamanan ke semua response: XSS Protection, no-sniff, no-frame, dan HSTS untuk mencegah serangan berbasis browser.

Header Security
02
🌍

Geo-blocking

Mampu memblokir akses berdasarkan negara asal. Konfigurasi fleksibel — bisa diaktifkan kapan saja untuk membatasi akses dari wilayah tertentu yang mencurigakan.

Geo Filter
03
🕵️

IP Reputation Check

Cek IP yang sudah terbukti jahat dari sesi sebelumnya. IP dengan reputasi buruk langsung ditolak tanpa perlu melewati layer pemeriksaan lainnya.

Reputation Filter
04
🛠️

User Agent Filter

Blokir tools hacking secara otomatis: sqlmap, Nikto, Nmap, Hydra, Metasploit, Burp Suite, dan tools berbahaya lainnya.

Tool Blocker
05
🤖

Bot Detection

Deteksi bot dan scraper dari pola header yang mencurigakan. Bot yang terdeteksi langsung dimasukkan ke blacklist otomatis tanpa intervensi manual.

Anti-Bot
06
💣

Payload Bomb Protection

Blokir request dengan body terlalu besar: lebih dari 1MB untuk auth atau lebih dari 10MB untuk semua endpoint, mencegah serangan payload bomb yang bisa melumpuhkan server.

Size Limit
07
💥

DDoS Protection

Max 200 request/menit per IP → blokir 30 menit. Max 100 request dalam 5 detik → blokir 2 jam. Dua threshold berbeda untuk menangani serangan ringan maupun berat.

Rate Limiting
08
🌊

Endpoint Flood Protection

Membatasi maksimal 30 request ke endpoint yang sama dalam 10 detik per IP, mencegah serangan flood yang menarget endpoint spesifik seperti login atau API kritis.

Endpoint Guard
09
🔑

Brute Force Protection

Khusus login/register — maksimal 5 percobaan gagal dalam 15 menit → blokir 1 jam + kirim email alert otomatis ke admin.

Auth Protection
10
⚖️

User Rate Limiter

Per user (JWT): 300 request/15 menit umum, 20 export/jam, dan 50 RAB/jam. Memastikan tidak ada user yang memonopoli resource server.

Per-User Limit
11
🛡️

CSRF Protection

Token one-time-use untuk semua request POST/PUT/DELETE (kecuali auth), mencegah serangan Cross-Site Request Forgery yang bisa memanipulasi aksi pengguna.

CSRF Guard
