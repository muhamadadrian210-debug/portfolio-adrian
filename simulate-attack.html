Cara Kerja

Bagaimana Firewall Merespons

1

Request Masuk

Setiap request yang masuk ke server dicegat oleh middleware firewall sebelum mencapai route handler manapun.

2

Cek Blacklist

IP address pengirim langsung dicek terhadap daftar blacklist permanen. Jika cocok, request langsung ditolak dengan kode 429.

3

Analisis Pola

Sistem menganalisis frekuensi request, pola error, dan kecepatan akses untuk mendeteksi perilaku mencurigakan.

4

Blokir Otomatis

Jika threshold terlampaui, IP diblokir otomatis dengan durasi yang disesuaikan tingkat keparahan serangan (30 menit hingga permanen).
